TripMD

Work done for medical software startup TripMD in New Delhi India. I interned there for 7 weeks in December 2014-January 2015 while completing the India China Insitute fellowship.

My main contribution to product development was conceptualizing and designing the SOS feature on the app. The feature allows the user to send personalized messages to predetermined contacts during a medical emergency with the click of a button.
